Testes do sistema gerenciado de estoques
----------------------------------------------

Para executar os testes é necessário o eclipse com a ferramenta Parteg instalada.

Para desenvolver os testes ainda é necessário a ferramenta Papyrs instalada no Eclipse ou Standalone.

Para ambos são necessárias duas dependencias que devem ser colocadas na raizdo projeto:

    selenium-server-standalone-2.53.0.jar
    selenium-java-2.53.0.jar
